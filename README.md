# The Knowledge House – Weekly Infographic Archive

This repository documents my weekly learning journey through The Knowledge House’s Cybersecurity Fellowship. Each week, I create structured notes and turn them into clean, visual infographics that summarize both the technical concepts and the career development lessons covered in class.

The goal of this repo is to:
- Build a consistent, public record of my progress
- Translate complex topics into clear, visual summaries
- Strengthen my technical communication skills
- Create a portfolio of work that reflects my growth over time

---

## 📚 Structure of This Repository

Each week includes:
- A **master markdown file** containing:
  - My raw notes
  - Source summaries
  - Infographic prompts
  - Reflections and takeaways
- One or more **infographics** generated from that week’s learning
- A link in the README for easy navigation

This keeps each week self‑contained and makes the repo easy to browse.

Navigation:
   - Each week includes a master markdown file with my notes, sources, prompts, and reflections, along with the infographics generated from that week’s learning.

## 🧠 Workflow Overview

Every week follows the same process:

1. **Collect Sources**  
   I gather all Cyber and Career learning materials for the week.

2. **Create Structured Notes**  
   I break down the content into clear, digestible sections inside the weekly markdown.

3. **Design Infographic Prompts**  
   I write prompts that map each source to the correct panel and define the layout.

4. **Generate Infographics**  
   I produce clean, modern visuals summarizing the week’s learning.

5. **Publish + Reflect**  
   I post the infographics on LinkedIn and store everything here for long‑term reference.

This workflow helps me stay consistent, organized, and intentional about how I learn.
