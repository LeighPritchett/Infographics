# This Week I Learned – Week 5: Active Directory Foundations

## Overview
Active Directory (AD) is Microsoft’s directory service used to manage identities, authentication, and access across a Windows environment.

This week, I worked through **TryHackMe** and **Microsoft Learn** labs covering:

- AD structure: forests, trees, domains, OUs  
- Domain Controllers and their role in authentication  
- Users, groups, and permission models  
- Group Policy Objects (GPOs) and how they apply (LSDOU)  
- Kerberos authentication flow  
- Basic AD security concepts: least privilege, password policies, account lockout policies  

### Hands‑On Tasks Included
- Creating users and groups  
- Assigning permissions  
- Exploring GPO behavior  
- Understanding how AD organizes and secures enterprise environments  

**Key takeaway:**  
AD is the backbone of identity and access in Windows networks, and understanding its structure is essential for both system administration and cybersecurity.
